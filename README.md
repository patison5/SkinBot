# SkinBot

### installation

Create config.js in root directory and paste there this code.

```js
const config = {
	API_KEY:     "YOUR_API_KEY",
	SECRET_HASH: "YOUR_SECRET_HASH"
}

module.exports = config;
```

Run the following command
```bash
$ npm install
```


To start server
```bash
$ node app.js
```
